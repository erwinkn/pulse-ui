<PyFunction name={"factorial"} type={"(n) -> int"}>

Calculate factorial recursively.

<PySourceCode >

```python
def factorial(n: int) -> int:
	"""Calculate factorial recursively."""
	if n <= 1:
		return 1
	return n * factorial(n - 1)
```

<PyFunction name={"clamp"} type={"(value, min_val, max_val) -> float"}>

Clamp value between min_val and max_val.

<PySourceCode >

```python
def clamp(value: float, min_val: float, max_val: float) -> float:
	"""Clamp value between min_val and max_val."""
	if value < min_val:
		return min_val
	if value > max_val:
		return max_val
	return value
```
