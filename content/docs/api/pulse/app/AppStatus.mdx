---
title: AppStatus
---

Application lifecycle status.

## Attributes

<PyAttribute name={"created"} type={null} value={"0"}>

App instance created but not yet initialized.

</PyAttribute>

<PyAttribute name={"initialized"} type={null} value={"1"}>

App.setup() has been called, routes configured.

</PyAttribute>

<PyAttribute name={"running"} type={null} value={"2"}>

App is actively serving requests.

</PyAttribute>

<PyAttribute name={"draining"} type={null} value={"3"}>

App is shutting down, draining connections.

</PyAttribute>

<PyAttribute name={"stopped"} type={null} value={"4"}>

App has been fully stopped.

</PyAttribute>
