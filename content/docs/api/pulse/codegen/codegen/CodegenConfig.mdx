---
title: CodegenConfig
---

Configuration for code generation output paths.

Controls where generated React Router files are written. All paths
can be relative (resolved against base_dir) or absolute.

## Attributes

<PyAttribute name={"web_dir"} type={"Path | str"} value={"'web'"}>

Root directory for web output.

</PyAttribute>

<PyAttribute name={"pulse_dir"} type={"Path | str"} value={"'pulse'"}>

Subdirectory name for generated files.

</PyAttribute>

<PyAttribute name={"base_dir"} type={"Path | None"} value={"None"}>

Explicit base directory, if provided.

</PyAttribute>

<PyAttribute name={"resolved_base_dir"} type={"Path"} value={null}>

Resolve the base directory where relative paths should be anchored.

<Callout title={"Resolution precedence"} type={"resolution-precedence"}>

1. Explicit `base_dir` if provided
2. Directory of PULSE_APP_FILE env var
3. PULSE_APP_DIR env var
4. Current working directory

</Callout>

</PyAttribute>

<PyAttribute name={"web_root"} type={"Path"} value={null}>

Absolute path to the web root directory.

</PyAttribute>

<PyAttribute name={"pulse_path"} type={"Path"} value={null}>

Full path to the generated Pulse app directory.

</PyAttribute>
