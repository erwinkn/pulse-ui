---
title: StableEntry
---

Container for a stable value and its wrapper function.

Holds a value and a wrapper function that always delegates to the
current value, allowing the wrapper reference to remain stable while
the underlying value can change.

## Attributes

<PyAttribute name={"__slots__"} type={null} value={"('value', 'wrapper')"} />

<PyAttribute name={"value"} type={"Any"} value={"value"}>

The current wrapped value.

</PyAttribute>

<PyAttribute name={"wrapper"} type={"Callable[..., Any]"} value={"wrapper"}>

Stable function that delegates to the current value.

</PyAttribute>

## Functions

<PyFunction name={"__init__"} type={"(self, value) -> None"}>

<PySourceCode >

```python
def __init__(self, value: Any) -> None:
	self.value = value

	def wrapper(*args: Any, **kwargs: Any):
		current = self.value
		if callable(current):
			return current(*args, **kwargs)
		return current

	self.wrapper = wrapper
```
