JavaScript Math builtin module.

Usage:

```python
import pulse.js.math as Math
Math.PI           # -> Math.PI
Math.floor(3.7)   # -> Math.floor(3.7)

from pulse.js.math import PI, floor
PI                # -> Math.PI
floor(3.7)        # -> Math.floor(3.7)
```
