title: weakmap

JavaScript WeakMap builtin module.

Usage:

```python
from pulse.js import WeakMap
WeakMap()                     # -> new WeakMap()
WeakMap([[obj, "value"]])    # -> new WeakMap([[obj, "value"]])

# Or import from module directly:
from pulse.js.weakmap import WeakMap
```
