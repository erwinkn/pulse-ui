JavaScript WeakSet builtin module.

Usage:

```python
from pulse.js import WeakSet
WeakSet()                     # -> new WeakSet()
WeakSet([obj1, obj2])        # -> new WeakSet([obj1, obj2])

# Or import from module directly:
from pulse.js.weakset import WeakSet
```
