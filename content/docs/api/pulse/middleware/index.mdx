<PyFunction name={"stack"} type={"(*middlewares) -> PulseMiddleware"}>

Compose multiple middlewares into a single middleware stack.

Example:

```python
import pulse as ps

app = ps.App(
    middleware=ps.stack(
        AuthMiddleware(),
        LoggingMiddleware(),
    )
)
```

<PySourceCode >

def stack(*middlewares: PulseMiddleware) -> PulseMiddleware:
	"""Compose multiple middlewares into a single middleware stack.

	Args:
		*middlewares: Middleware instances to compose.

	Returns:
		``MiddlewareStack`` instance.

	Example:

	\```python
	import pulse as ps

	app = ps.App(
	    middleware=ps.stack(
	        AuthMiddleware(),
	        LoggingMiddleware(),
	    )
	)
	\```
	"""
	return MiddlewareStack(list(middlewares))
```
