---
title: Page
---

Named tuple representing a page in an infinite query.

Each page contains the fetched data and the parameter used to fetch it,
enabling cursor-based or offset-based pagination.

## Attributes

<PyAttribute name={"data"} type={"T"} value={null}>

The fetched page data of type T.

</PyAttribute>

<PyAttribute name={"param"} type={"TParam"} value={null}>

The page parameter (cursor, offset, etc.) used to fetch this page.

</PyAttribute>
