title: RouteContext
---

Runtime context for the current route.

Provides reactive access to the current route's URL components and
parameters. Accessible via `ps.route()` in components.

## Attributes

<PyAttribute name={"info"} type={"RouteInfo"} value={"cast(RouteInfo, cast(object, ReactiveDict(info)))"}>

Current route info (reactive, auto-updates on navigation).

</PyAttribute>

<PyAttribute name={"pulse_route"} type={"Route | Layout"} value={"pulse_route"}>

Route or Layout definition for this context.

</PyAttribute>

<PyAttribute name={"pathname"} type={"str"} value={null}>

Current URL path (e.g., "/users/123").

</PyAttribute>

<PyAttribute name={"hash"} type={"str"} value={null}>

URL hash fragment (without #).

</PyAttribute>

<PyAttribute name={"query"} type={"str"} value={null}>

Raw query string (without ?).

</PyAttribute>

<PyAttribute name={"queryParams"} type={"dict[str, str]"} value={null}>

Parsed query parameters as dict.

</PyAttribute>

<PyAttribute name={"pathParams"} type={"dict[str, str]"} value={null}>

Dynamic path parameters (e.g., \{"id": "123"\} for ":id").

</PyAttribute>

<PyAttribute name={"catchall"} type={"list[str]"} value={null}>

Catch-all segments as list.

</PyAttribute>
