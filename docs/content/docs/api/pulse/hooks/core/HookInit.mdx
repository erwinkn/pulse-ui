title: "HookInit"
---

Initialization context passed to hook factories.

When a hook factory accepts a single argument, it receives this object
containing context about the initialization.

## Attributes

<PyAttribute name={"key"} type={"str | None"} value={null}>

Optional key if the hook was called with a specific key.

</PyAttribute>

<PyAttribute name={"render_cycle"} type={"int"} value={null}>

The current render cycle number.

</PyAttribute>

<PyAttribute name={"definition"} type={"Hook[T]"} value={null}>

Reference to the Hook definition being initialized.

</PyAttribute>
