title: "HookMetadata"

Metadata for a registered hook.

Contains optional descriptive information about a hook, useful for
debugging and documentation.

## Attributes

<PyAttribute name={"description"} type={"str | None"} value={"None"}>

Human-readable description of the hook's purpose.

</PyAttribute>

<PyAttribute name={"owner"} type={"str | None"} value={"None"}>

Module or package that owns this hook (e.g., "pulse.core").

</PyAttribute>

<PyAttribute name={"version"} type={"str | None"} value={"None"}>

Version string for the hook implementation.

</PyAttribute>

<PyAttribute name={"extra"} type={"Mapping[str, Any] | None"} value={"None"}>

Additional metadata as a mapping.

</PyAttribute>
