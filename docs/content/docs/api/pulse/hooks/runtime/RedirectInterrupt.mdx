---
title: "RedirectInterrupt"
---

Exception raised to interrupt render and trigger a redirect.

This exception is thrown by ``ps.redirect()`` to interrupt the current
render cycle and navigate to a different path.

## Attributes

<PyAttribute name={"path"} type={"str"} value={"path"}>

The destination URL to redirect to.

</PyAttribute>

<PyAttribute name={"replace"} type={"bool"} value={"replace"}>

If True, replaces the current history entry instead of pushing.

</PyAttribute>
