---
title: "ActionError"
---

Failed query action result.

Returned by query operations like ``refetch()`` and ``wait()`` when the
operation fails after exhausting retries.

## Attributes

<PyAttribute name={"error"} type={"Exception"} value={null}>

The exception that caused the failure.

</PyAttribute>

<PyAttribute name={"status"} type={"Literal['error']"} value={"'error'"}>

Always ``"error"`` for discriminated union matching.

</PyAttribute>
