---
title: "Chat with Channels"
---

Server side:

```python
chat = ps.channel("chat")

@chat.on("message")
def on_message(payload):
    chat.broadcast("new_message", payload)
```

Client side (JS client):

```ts
const ch = usePulseChannel('chat')
ch.on('new_message', (msg) => ...)
ch.emit('message', { text })
```
