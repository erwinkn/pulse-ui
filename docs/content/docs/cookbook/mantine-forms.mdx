---
title: "Mantine Forms"
---

Use `MantineForm` from `pulse-mantine`.

```python
form = ps.setup(lambda: MantineForm(initialValues={"email": ""}))
return form.render(onSubmit=handle)[
    TextInput(name="email", label="Email"),
    Button("Submit", type="submit"),
]
```
