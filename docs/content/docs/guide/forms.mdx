---
title: "Forms"
---

`ps.Form` registers server-side forms.

```python
ps.Form(key="demo", onSubmit=handle)[
    ps.input(name="email", type="email"),
    ps.button("Submit", type="submit"),
]
```

`ManualForm` exposes `.props()` for custom wiring.
`FormData` supports file uploads via `UploadFile`.
