---
title: "JS Interop"
---

Use `@ps.javascript` to transpile Python to JS.

```python
from pulse.js import document

@ps.javascript
def focus(sel: str):
    document.querySelector(sel).focus()
```

Use `ps.run_js` inside callbacks to execute on the client.
