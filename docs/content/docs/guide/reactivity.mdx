---
title: "Reactivity"
---

Signals and computeds power change tracking.

On `State`:
- `@ps.computed` creates cached derived values
- `@ps.effect` runs on dependency changes

Use `ps.Batch()` to group updates and `ps.Untrack()` to avoid tracking.
