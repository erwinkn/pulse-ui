title: "Routing"
---

Routes are defined on the `App`.

```python
app = ps.App([
    ps.Route("/", home),
    ps.Layout("/app", layout, children=[
        ps.Route("/settings", settings),
    ]),
])
```

Client helpers:
- `Link`, `Outlet`
- `ps.navigate` for programmatic navigation
