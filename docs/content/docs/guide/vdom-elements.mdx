---
title: "VDOM and Elements"
---

Pulse elements are Python functions that describe DOM.

```python
ps.div(
    ps.h1("Title"),
    ps.p("Body"),
    className="p-6",
)
```

Props follow React naming (`className`, `htmlFor`).
Children can be positional or bracket syntax:

```python
ps.div(className="...")[
    ps.h1("Title"),
    ps.p("Body"),
]
```
