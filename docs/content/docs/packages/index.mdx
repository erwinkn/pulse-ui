Pulse provides optional packages that extend the core framework with UI components, charts, icons, authentication, and deployment tools.

## UI Components

### pulse-mantine

Full bindings for [Mantine](https://mantine.dev/), a modern React component library. Includes 100+ components, a form system with client-side validation, date pickers, and charts.

```bash
uv add pulse-mantine
```

[Read more](/docs/packages/pulse-mantine)

### pulse-ag-grid

Bindings for [AG Grid](https://www.ag-grid.com/), a feature-rich data grid component. Ideal for displaying and editing tabular data.

```bash
uv add pulse-ag-grid
```

[Read more](/docs/packages/pulse-ag-grid)

### pulse-recharts

Bindings for [Recharts](https://recharts.org/), a composable charting library. Use this for custom chart configurations beyond what Mantine Charts provides.

```bash
uv add pulse-recharts
```

[Read more](/docs/packages/pulse-recharts)

### pulse-lucide

Bindings for [Lucide](https://lucide.dev/) icons. Provides 1000+ customizable icons as Pulse components.

```bash
uv add pulse-lucide
```

[Read more](/docs/packages/pulse-lucide)

## Core

### pulse-client

The JavaScript client library that powers the browser side of Pulse applications. Handles WebSocket communication, VDOM rendering, and event serialization.

[Read more](/docs/packages/pulse-js-client)

## Authentication

### pulse-msal

Microsoft Entra ID (Azure AD) authentication plugin using MSAL (Microsoft Authentication Library).

```bash
uv add pulse-msal
```

[Read more](/docs/packages/pulse-msal)

## Deployment

### pulse-aws

Deploy Pulse applications to AWS ECS Fargate with a single command.

```bash
uv add pulse-aws
```

[Read more](/docs/packages/pulse-aws)
