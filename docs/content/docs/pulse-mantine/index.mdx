Python bindings for [Mantine](https://mantine.dev/), a modern React component library with 100+ components, a form system, date pickers, and charts.

## Installation

```bash
uv add pulse-mantine
```

## Quick Start

Wrap your app with `MantineProvider` to enable Mantine styling:

```python
import pulse as ps
from pulse_mantine import MantineProvider, Button, TextInput, Stack

@ps.component
def App():
    return MantineProvider()[
        Stack(gap="md")[
            TextInput(label="Name", placeholder="Enter your name"),
            Button("Submit"),
        ]
    ]

app = ps.App([ps.Route("/", App)])
```

## Forms

Pulse Mantine includes a form system with client-side validation that runs in the browser for instant feedback.

### Basic Form

```python
import pulse as ps
from pulse_mantine import (
    MantineProvider, MantineForm, Button, TextInput, Checkbox,
    Stack, Group, IsEmail, IsNotEmpty,
)

@ps.component
def SignupForm():
    with ps.init():
        form = MantineForm(
            initialValues={
                "email": "",
                "password": "",
                "terms": False,
            },
            validate={
                "email": IsEmail("Please enter a valid email"),
                "password": IsNotEmpty("Password is required"),
            },
        )

    def handle_submit(values):
        print("Form submitted:", values)

    return MantineProvider()[
        form.render(onSubmit=handle_submit)[
            Stack(gap="md")[
                TextInput(
                    name="email",
                    label="Email",
                    placeholder="you@example.com",
                    withAsterisk=True,
                ),
                TextInput(
                    name="password",
                    label="Password",
                    type="password",
                    withAsterisk=True,
                ),
                Checkbox(
                    name="terms",
                    label="I agree to the terms of service",
                ),
                Group(justify="flex-end")[
                    Button("Sign Up", type="submit"),
                ],
            ]
        ]
    ]

app = ps.App([ps.Route("/", SignupForm)])
```

### Validators

Built-in validators for common use cases:

```python
from pulse_mantine import (
    # String validators
    IsNotEmpty,       # Value is not empty
    IsEmail,          # Valid email format
    Matches,          # Matches regex pattern
    HasLength,        # String length constraints
    StartsWith,       # Starts with prefix
    EndsWith,         # Ends with suffix

    # Number validators
    IsNumber,         # Is a number
    IsInteger,        # Is an integer
    IsInRange,        # Within min/max range

    # Date validators
    IsDate,           # Is a valid date
    IsBefore,         # Before a date
    IsAfter,          # After a date

    # Other validators
    IsUrl,            # Valid URL
    IsUUID,           # Valid UUID
    MatchesField,     # Matches another field
)
```

### Accessing Form Values

Get form values programmatically:

```python
@ps.component
def FormWithAccess():
    with ps.init():
        form = MantineForm(initialValues={"name": ""})

    async def get_values():
        values = await form.get_form_values()
        print("Current values:", values)

    return form.render()[
        TextInput(name="name", label="Name"),
        Button("Get Values", onClick=get_values),
    ]
```

## Date Pickers

Date and time components from `@mantine/dates`:

```python
from pulse_mantine import (
    DateInput, DatePicker, DatePickerInput,
    DateTimePicker, TimeInput, Calendar,
    MonthPicker, YearPicker,
)

@ps.component
def DateDemo():
    with ps.init():
        form = MantineForm(initialValues={"date": None})

    return MantineProvider()[
        form.render()[
            DatePickerInput(
                name="date",
                label="Select a date",
                placeholder="Pick a date",
            ),
        ]
    ]
```

## Charts

Charts powered by `@mantine/charts` (built on Recharts):

```python
from pulse_mantine import (
    MantineProvider, LineChart, AreaChart, BarChart, PieChart,
    DonutChart, RadarChart, ScatterChart,
)

data = [
    {"month": "Jan", "sales": 4000, "expenses": 2400},
    {"month": "Feb", "sales": 3000, "expenses": 1398},
    {"month": "Mar", "sales": 2000, "expenses": 9800},
    {"month": "Apr", "sales": 2780, "expenses": 3908},
]

@ps.component
def ChartDemo():
    return MantineProvider()[
        LineChart(
            h=300,
            data=data,
            dataKey="month",
            series=[
                {"name": "sales", "color": "blue"},
                {"name": "expenses", "color": "red"},
            ],
            withLegend=True,
            withTooltip=True,
        )
    ]

app = ps.App([ps.Route("/", ChartDemo)])
```

## Notifications

Display toast notifications:

```python
from pulse_mantine import MantineProvider, Button, notifications

@ps.component
def NotificationDemo():
    def show_notification():
        notifications.show(
            title="Success",
            message="Your changes have been saved",
            color="green",
        )

    return MantineProvider()[
        Button("Show Notification", onClick=show_notification),
    ]
```

## App Shell

Build application layouts with header, navbar, and aside:

```python
from pulse_mantine import (
    MantineProvider, AppShell, AppShellHeader, AppShellNavbar,
    AppShellMain, NavLink, Text,
)

@ps.component
def Dashboard():
    return MantineProvider()[
        AppShell(
            header={"height": 60},
            navbar={"width": 250, "breakpoint": "sm"},
            padding="md",
        )[
            AppShellHeader()[
                Text("My App", size="lg", fw=700),
            ],
            AppShellNavbar()[
                NavLink(label="Home", href="/"),
                NavLink(label="Settings", href="/settings"),
            ],
            AppShellMain()[
                Text("Main content goes here"),
            ],
        ]
    ]
```
