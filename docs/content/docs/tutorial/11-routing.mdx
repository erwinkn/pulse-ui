---
title: "Routing"
---

Routes are defined on `App`.

```python
app = ps.App([
    ps.Route("/", home),
    ps.Layout("/app", layout, children=[
        ps.Route("/settings", settings),
    ]),
])
```

Client helpers:
- `Link`, `Outlet`
- `ps.navigate("/path")`
- `ps.route()` for route info
