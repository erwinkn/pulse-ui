---
title: "Sessions and Middleware"
---

Session data is per user.

```python
def page():
    session = ps.session()
    session["visits"] = (session.get("visits") or 0) + 1
```

Middleware can short-circuit.

```python
class Auth(ps.PulseMiddleware):
    async def connect(self, *, request, session, next):
        if not session.get("user"):
            return ps.Deny()
        return await next()
```
